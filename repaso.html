Posible solución

  <head>
    <meta charset="UTF-8">

        display: grid;

¿Qué se ha corregido y qué aporta?
  • Se ha añadido <!DOCTYPE html> y se ha cerrado la etiqueta <html>.
  • Se ha incluido la etiqueta <body> y se ha movido el contenido fuera del <head>.
  • Se han añadido etiquetas semánticas: <header>, <nav>, <main>, <section>, <h1>, <ul>, <li>, <footer>.
  • Se ha corregido la estructura general y se ha añadido un diseño con grid-area para organizar el layout.
  • El uso de grid-area permite un diseño flexible y semántico, separando claramente las áreas de la página.
  • Se han aplicado las clases main-content y content-section para un diseño visual coherente con el resto del proyecto.

Mejora de estructura con CSS Grid

La siguiente estructura utiliza CSS Grid para dividir la página en áreas bien definidas: encabezado, banner lateral, contenido principal, barra lateral y pie de página. Esto facilita la organización visual y la adaptabilidad del diseño.

Ejemplo de CSS

.grid-container {
    display: grid;
    grid-template-areas:
      "header header header"
      "banner main aside"
      "footer footer footer";
    grid-template-columns: 150px 1fr 220px;
    grid-template-rows: auto 1fr auto;
    min-height: 100vh;
    }

    header {
      grid-area: header;
    }
    .banner {
      grid-area: banner;
    }
    main {
      grid-area: main;
    }
    aside {
      grid-area: aside;
    }
    footer {
      grid-area: footer;
    }

Ejemplo de HTML

<div class="grid-container">
      <header>Encabezado</header>
      <div class="banner">Banner</div>
      <main>Contenido principal</main>
      <aside>Barra lateral</aside>
      <footer>Pie de página</footer>
    </div>
