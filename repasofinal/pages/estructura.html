Estructura de un documento html

Un documento HTML tiene una estructura básica que incluye varios elementos esenciales. Aquí te presento la estructura típica de un documento HTML:

<!DOCTYPE html>    <html lang="es">
  <head>        
    <meta charset="UTF-8">  
    <meta name="viewport" content="width=device-width, initial-scale=1.0">  
    <title>Título de la página</title>  
    <link rel="stylesheet" href="styles.css">  <!-- Enlace a archivo CSS externo -->
    <script src="script.js" defer></script>  <!-- Enlace a archivo JavaScript externo --> 
    
    </head>
