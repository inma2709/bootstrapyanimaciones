Página montada con Grid — 3 zonas (Header, Contenido y Footer)

Esta página completa es un Grid con 3 áreas verticales. Todo el paso a paso y los ejemplos están dentro del <main>.

Explicación paso a paso

Paso 1 Activar Grid en el contenedor (body)

Convertimos el <body> en un contenedor Grid con display: grid.

/* CSS */
body{
  display: grid;     /* ← activa CSS Grid en el body */
}

Paso 2 Definir el mapa de áreas (3 zonas)

Declaramos 3 filas (una por línea) y una sola columna: header, content y footer.

/* CSS */
body{
  grid-template-areas:
    "header"
    "content"
    "footer";
}

Paso 3 Definir columnas y filas (fr y auto)

Usamos 1fr para la única columna (flexible) y auto 1fr auto para filas: header/foot se ajustan al contenido y la zona central crece. 

/* CSS */
body{
  grid-template-columns: 1fr;  /* 1 columna flexible */
  grid-template-rows: auto 1fr auto;  /* 3 filas: header (auto), content (1fr), footer (auto) */
}

Recuerda: fr reparte el espacio disponible tras asignar tamaños fijos/auto.

Paso 4 Asignar cada hijo a su área (grid-area)

Solo los hijos directos del body pueden engancharse a las áreas con grid-area.

/* CSS */
header { grid-area: header; }
main   { grid-area: content; }
footer { grid-area: footer; }
<!-- HTML mínimo -->
<body>
  <header>Cabecera</header>
  <main>Contenido</main>
  <footer>Pie</footer>
</body>

Importante: el HTML no necesita clases especiales; la colocación la decide CSS.

Paso 5 Gaps y respiración

gap separa las áreas; padding añade espacio interior al contenedor.

/* CSS */
body{
  gap: 16px;     /* separa filas/columnas del grid */
  padding: 16px; /* espacio interior del body */
}

Paso 6 (Opcional) Adaptar en responsive cambiando el mapa

Puedes redefinir el mapa en media queries sin tocar el HTML. Aquí mantenemos 1 columna, pero podrías cambiar a varias si más adelante añades zonas laterales. 

/* CSS */
@media (min-width: 960px){
  body{
    /* mismo HTML, podrías cambiar columnas/filas si lo necesitas */
    grid-template-columns: 1fr;
  }
}

Resumen Página completa con Grid (3 zonas)

Este es el esqueleto mínimo funcional que estás viendo en esta misma página:

/* CSS esencial */
body{
  display: grid;
  grid-template-areas:
    "header"
    "content"
    "footer";
  grid-template-columns: 1fr;
  grid-template-rows: auto 1fr auto;
  gap: 16px;
  min-height: 100vh;
  padding: 16px;
}
header{ grid-area: header; }
main{   grid-area: content; }
footer{ grid-area: footer; }
<!-- HTML esencial -->
<body>
  <header>...</header>
  <main>...</main>
  <footer>...</footer>
</body>
