¿Qué es CSS Grid?

CSS Grid es un sistema de diseño bidimensional que te permite organizar elementos en filas y columnas. Es ideal para crear layouts complejos y responsivos, como el esqueleto de una página web.

¿Por qué usar Grid? Porque te da control total sobre la posición de cada elemento, permitiendo crear desde simples galerías hasta dashboards avanzados.

  • Define áreas de la página (header, nav, main, aside, footer) de forma clara.
  • Permite superponer elementos y crear diseños asimétricos.
  • Facilita la adaptación a diferentes tamaños de pantalla.
¿Cómo se declara Grid? 
/* 1) Convertimos el contenedor en grid */
.container { display: grid; }

/* 2) Definimos columnas y filas */
.container {
  grid-template-columns: 1fr 2fr 1fr;    /* 3 columnas */
  grid-template-rows: auto auto;         /* 2 filas */
  gap: 12px;                             /* separación */
}

/* 3) Colocamos cada ítem */
.header { grid-column: 1 / -1; }         /* ocupa las 3 columnas */
.main   { grid-column: 1 / 3; }          /* columnas 1 a 2 */
.aside  { grid-column: 3; }              /* columna 3 */
.footer { grid-column: 1 / -1; }

Recuerda: 1fr reparte el espacio disponible. -1 es el último borde de la rejilla.

Consejo: Usa grid-template-areas para nombrar zonas y hacer tu CSS más legible.

¿Qué es Flexbox?

Flexbox es un sistema de diseño unidimensional: organiza elementos en una fila o columna, alineándolos y distribuyéndolos de forma eficiente. Es perfecto para menús, barras de navegación, tarjetas en fila o para centrar elementos.

  • Ideal para componentes donde el orden y el tamaño de los elementos puede variar.
  • Permite centrar fácilmente tanto horizontal como verticalmente.
  • Facilita el diseño responsivo de elementos en fila o columna.
¿Cómo se declara Flexbox? 
/* 1) Convertimos el contenedor en flex */
.container { display: flex; }

/* 2) Elegimos la dirección */
.container { flex-direction: row; }   /* fila (por defecto) */
/* .container { flex-direction: column; }   columna */

/* 3) Alineamos y distribuimos */
.container {
  justify-content: center;   /* eje principal (x si row) */
  align-items: center;       /* eje cruzado  (y si row) */
  gap: 12px;                 /* hueco entre ítems */
}

Recuerda: justify-content actúa sobre el eje principal y align-items sobre el eje cruzado. El eje cambia según uses row o column.

Tip: Flexbox es tu mejor amigo para centrar cosas. ¡Pruébalo!

¿Cuándo usar cada uno?

  • Grid: para layouts generales, galerías, dashboards, y estructuras 2D.
  • Flex: para menús, barras, tarjetas en fila/columna, y centrados rápidos.
  • Combinados: usa Grid para la estructura principal y Flex para alinear el contenido interno.
Ejemplo: Una página puede usar Grid para el layout general y Flexbox para el menú de navegación.

Chuleta rápida

Flex: propiedades clave (contenedor)

  • display: flex | inline-flex
  • flex-direction: row | column | …
  • justify-content: start | center | space-between | …
  • align-items: stretch | center | …
  • gap: 12px, flex-wrap: wrap

Flex: propiedades de los hijos

  • flex: 1 (crece/reduce)
  • align-self: center (alineación individual)
  • order: 2 (reordenar)

Grid: lo esencial

  • display: grid
  • grid-template-columns: repeat(3, 1fr)
  • grid-template-rows: auto auto
  • gap: 12px
  • grid-column / grid-row para colocar ítems
